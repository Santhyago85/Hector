Resumo da vaga

Desenvolver uma aplicação backend RESTful com dois endpoints. O primeiro endpoint fará a autenticação e devolverá um token e o segundo usará esse token para salvar os dados enviados.

Requisitos para o cargo

Conhecimento intermediário em JAVA.

Conhecimento intermediário em Linux.

  Experiência mínima de 9 meses na área.

Desejável

MEAN(Mongo/Express/Angular/Node).

Salesforce.

  Asterisk.

Como participar

O candidato deve criar uma aplicação em node ou java e deixar disponível na internet para ser acessada pela equipe técnica conforme as especificações. A equipe técnica da G4Flex irá autentificar e usar o token para enviar um json que conterá as informações para a última fase que deverão ser salvo em um banco de dados.

Critérios avaliados

Autenticação via token.

Uso da Tecnologia RESTful.

  Código Limpo.
